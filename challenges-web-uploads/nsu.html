Project Details

Project background and rationale

  • speed up creation of lesson plans
  • consistent data entry
  • linkages with other services (ex. Instruction request service)

Headless CMS & React

  • WordPress as a Headless CMS
  • WordPress REST API
  • WP Plug-in - Advanced Custom Fields (ACF) & ACF to REST API integration
  • React front end

Development Tools

  • WordPress
  • IDE - VSCode
  • Postman - REST Client
  • React DEV tools - chrome extension
  • GitHub - project repository

Data Model

Authentication

  • Install JWT Authentication WordPress plugin to use JSON Web Tokens (JWT)
  • Update .htaccess file to allow HTTP Authorization Headers
  • Front End - asynchronous POST request - return a 'bearer token' and save it into a cookie

Working with the Lessons List & Lesson Detail

  • All Lessons - REST API GET query
  • Load data into state using Context API

New Lesson Form

  • REST API queries for linked data (ex. Instructors, courses, concepts)
  • Build lesson - maintain data in state with Context API
  • Drag and Drop - module dependency React DnD
  • Validation - (ex. required fields before button for next step is active)
  • Save - POST query with body = form data is JSON format

UI

  • TailwindCSS
  • Responsive layout (ex. home page, new lesson form)
